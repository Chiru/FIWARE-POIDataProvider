WEB-72

Subfolders:

  orion_adapter
    A cron job to feed Orion Context Broker with data from vehicle_poi_server.
  orion_enhanced_poi_server
    A POI server that provides POI data from Orion Context Broker and local
    POI data base.
  poi_clients
    Web pages to show poi data.
  vehicle_poi_server
    Server that provides vehicle data from the servers of public transportation
    services of the cities of Helsinki and Tampere, as POI's.

Each subfolder contains its own readme.txt.
